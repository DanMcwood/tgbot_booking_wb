**WB Booking Telegram Bot**

Telegram-бот для автоматического бронирования слотов на Wildberries.
Проект демонстрирует навыки разработки Telegram-ботов на Python, асинхронную архитектуру, работу с API Wildberries, работу с анти-детект бразурами, вкладками и управление пользователями.

**Описание проекта**

WB Booking Bot — это система, которая помогает продавцам Wildberries автоматизировать бронирование свободных слотов поставки.
Бот работает через Telegram, предоставляет удобный интерфейс для настройки параметров: склада, типа поставки, коэффициента, даты и частоты поиска слотов. При появлении подходящего слота бот может сразу забронировать его и уведомить пользователя.

**Функциональность**

Регистрация пользователей через Telegram (Telegram ID + дополнительные настройки).

**Настройка параметров бронирования:**

выбор склада Wildberries, добавление избранных складов

выбор типа поставки

установка целевого коэффициента слотов

настройка диапазонов дат для поиска

Автоматический поиск слотов с заданными параметрами.

Автоматическое бронирование — бот может забронировать слот, как только он появляется.

Уведомления: бот присылает сообщения о статусе — “слот найден”, “слот забронирован”, “ошибка” и т.д.

Хранение настроек пользователей и истории бронирований в базе данных.

/бот регулярно проверяет по апи, появился ли свободный слот для бронирования, при положительном отввете через браузер бронирует поставку с нужными параметрами

**Стэк**

Проект использует широкий стек библиотек Python для разных задач:

aiogram, python-telegram-bot, telebot — работа с Telegram API

aiohttp, httpx, websockets, trio-websocket, websocket-client — асинхронные HTTP и WebSocket соединения

aiosqlite, sqlite, mysql-connector-python — база данных

selenium, undetected-chromedriver, selenium-stealth, playwright, playwright-stealth — автоматизация браузера и работы с Wildberries

pandas, openpyxl, gspread — работа с Excel и Google Sheets

python-dotenv — хранение конфиденциальных данных в .env

cryptography, fernet, pycryptodome — шифрование данных

beautifulsoup4, lxml, bs4, soupsieve — парсинг HTML

fastapi, Flask, bottle, bottle-websocket, Eel — веб-интерфейсы и API

loguru, colorlog — логирование

aiocron, croniter — планировщик задач

pydantic, annotated-types — валидация и структуры данных

и многие другие — вспомогательные и системные библиотеки для корректной работы всех модулей проекта

**Преимущества и особенности**

Высокая скорость реагирования — асинхронные запросы к API WB позволяют быстро обнаруживать слоты.

Пользовательский интерфейс через Telegram — удобно настраивать параметры прямо через чат.

Автоматизация — бот действует без необходимости постоянного участия пользователя.

Гибкость — пользователь может задать диапазоны дат, коэффициенты, типы поставок и т.п.

Безопасность — конфиденциальные API-данные пользователей могут быть защищены.
